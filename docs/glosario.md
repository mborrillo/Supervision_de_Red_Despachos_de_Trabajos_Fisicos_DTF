# Glosario de métricas y campos

- **ADECIR**: Identificador único del despacho de trabajo físico.
- **FECHA_ENVIO**: Fecha en que se genera/envía el despacho.
- **FECHA_RES**: Fecha en que se marca como finalizado.
- **DIAS_FINALIZADOS**: días entre FECHA_ENVIO y FECHA_RES (solo finalizados).
- **DIAS_GESTION**: días entre FECHA_ENVIO y la fecha de corte (solo abiertos).
- **TIEMPOS_DG**: EN OBJETIVO / FUERA OBJETIVO según SLA de 4 días para finalizados.
- **TIEMPOS_DP**: EN OBJETIVO / FUERA OBJETIVO para abiertos (concepto histórico).
- **AOP_RESP**: Región operativa derivada de Provincia (AMBA, AOP SUR, etc.).
- **Operacion**: Clasificación de Tipo de Movimiento (ALTA, BAJA, CAMBIO VEL/TECNICO, OTROS).
